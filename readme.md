# Dembed

## What Dembed can do ~

- Can send Rich Embed
- Can send premade Embed
- Tons of Commands added
- Can send custom embed message



## Installation

Dembed requires [Node.js](https://nodejs.org/) v12+ to run.

Install now.

```sh
npm i dembed@latest
```

## Start using dembed

```sh
var embed = require("dembed")
var embed = new embed.Dembed({ customize: true });
```# dembed
